# library-simulator

University project members: Daniel Brauła and Bartosz Jasiński


A project created using an object-oriented programming approach in C++. When the code is run, the bookshop is controlled via a console in the terminal. The user, in order to control the bookshop, uses numbers from 0-9 or types in the specific items that interest him/her after an appropriate message displayed on the console in the terminal.
